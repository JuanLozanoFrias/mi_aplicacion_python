# Cotizador Weston (DEMO)

Como correr
- Ejecuta `python main.py` (o `start_dev.bat` si usas el launcher local).
- En el menu lateral elige `COTIZADOR` (badge DEMO).

JSON requeridos
- `data/cotizador/catalogo_demo.json`
- `data/cotizador/reglas_demo.json`
- `data/cotizador/clientes_demo.json`
- `data/cotizador/proyecto_demo.json`

Exportar PDF
- En el modulo, usa `EXPORTAR PDF`.
- El archivo queda en `exports/cotizaciones/` con timestamp en el nombre.
